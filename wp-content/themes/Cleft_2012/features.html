960 Grid Based Framework

The 12-column grid is divided into portions that are 60 pixels wide. Each column has 10 pixels of margin on the left and right, which create 20 pixel wide gutters between columns. Examples!

Simply link to an image to activate the lightbox!

Group similar photos in a gallery with rel="lightbox"
<a rel="lightbox" href="full-size-image.jpg" title="Optional Title">Lightbox Link</a>
